# 🎨 Gerador de Página

Um criador de landing pages inteligente que utiliza **IA** para gerar sites completos e visualmente impressionantes a partir de um simples prompt!

## ✨ Funcionalidades

- 🤖 **Geração por IA**: Utilize a API Groq para gerar landing pages profissionais
- 🎯 **Prompt Simples**: Descreva seu negócio e a IA cria o site completo
- 💻 **HTML + CSS Puro**: Código limpo e otimizado
- 📱 **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- 🚀 **Instantâneo**: Obtenha seu site pronto em segundos

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com gradientes, animações e flexbox
- **JavaScript** - Lógica de requisição e integração com API
- **Groq API** - Inteligência artificial para geração de conteúdo
- **Responsive Design** - Mobile-first approach
